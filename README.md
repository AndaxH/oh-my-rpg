# Oh My RPG

RPG logic for web RPGs in TypeScript.

## Installation
```bash
nvm use 8
npm install --global lerna

lerna clean
lerna exec -- rm -f package-lock.json
rm -f package-lock.json
npm install
lerna run install

lerna bootstrap
```
